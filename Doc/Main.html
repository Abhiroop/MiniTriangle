Usage (command line):

hmtc [options\
file.mt] Compile "file.mt"
hmtc [options\
] Read program to compile from standard input. (Could be confusing!)

Options:

--help
Print help message and stop.
--stop-after-scanning
Stop after scanning. Implies "--print-after-scanning".
--print-after-scanning
Print intermediate representation after scanning.
--stop-after-parsing
Stop after parsing. Implies "--print-after-parsing".
--print-after-parsing
Print intermediate representation after parsing.
--stop-after-checking
Stop after type checking. Implies "--print-after-checking".
--print-after-checking
Print intermediate representation after type checking.
--version
Print HMTC version and stop.
